Data Mapping

Y-axis

Choose the attribute to be mapped to Y-axis from the data.
If the attribute is part of origin id, write just the attribute name
(eg. soure_id not origin_id.source_id) and check the checkbox below.
However if you want to group data to y-axis based on metada, then you
need to write "metada.attributename".

States

Define the states to be visualized. To open states write the name of each initial state you want to visualize.
To intermediate states write all states that are not initial or resolution states you whish to explore.
To resolution states write all resolution states you wich to use in visualization.
Note that the states are fetched from the event's type attribute!!
